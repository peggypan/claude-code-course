# Claude Code 源码解析 — 交互式课程

## 📚 课程介绍

这是一个为**非技术背景用户**设计的交互式课程，帮助你深入理解 **Claude Code**（AI 编程助手）的工作原理。

无论你是：
- 使用 AI 工具的"氛围程序员"
- 想要更好地理解 AI 助手如何工作
- 希望能与 AI 编程工具更有效地沟通

这门课程都适合你！

### 你将学到什么

✅ **模块 1：这是什么产品？**
- Claude Code 是什么，它如何帮助你
- 当你输入命令时，背后发生了什么
- 完整的请求流程追踪

✅ **模块 2：核心组件介绍**
- 查询引擎（QueryEngine）- 协调器
- 工具系统（Tools）- 执行器
- 状态管理（AppState）- 记忆库

✅ **模块 3：工具系统如何工作**
- 40+ 工具如何注册和协作
- 权限系统如何保护你的电脑
- 工具调用的完整生命周期

## 🚀 如何使用

### 方法 1：直接在浏览器中打开（推荐）

```bash
# 在课程目录中
open index.html
# 或者双击 index.html 文件
```

### 方法 2：使用本地服务器

```bash
# 使用 Python
python -m http.server 8000

# 或使用 Node.js
npx serve

# 然后在浏览器中访问
# http://localhost:8000
```

## 📖 学习提示

### 导航方式
- **滚动**：向下滚动浏览内容
- **点击圆点**：顶部导航栏的圆点可以快速跳转到不同模块
- **键盘**：使用 ↑↓ 或 ←→ 键在模块之间导航

### 交互元素
- **术语提示**：悬停或点击带虚线下划线的术语，查看解释
- **代码翻译**：左侧是真实代码，右侧是中文逐行解释
- **动画演示**：点击"下一步"按钮查看流程动画
- **群聊对话**：点击"播放全部"或"下一步"查看组件对话
- **测验**：选择答案后点击"检查答案"，查看反馈

### 学习建议
1. **按顺序学习**：模块从基础到进阶，依次学习效果最佳
2. **动手操作**：尝试与所有交互元素互动
3. **完成测验**：每个模块末尾都有测验，帮助你巩固知识
4. **随时回顾**：使用顶部导航快速回到任何模块

## 🔧 自定义和扩展

### 修改配色主题

编辑 `_base.html` 文件中的 CSS 变量：

```css
:root {
  --color-accent:       #D94F30;  /* 主色调 */
  --color-accent-hover: #C4432A;  /* 悬停颜色 */
  --color-accent-light: #FDEEE9;  /* 浅色背景 */
  --color-accent-muted: #E8836C;  /* 柔和色 */
}
```

可用配色方案：
- **vermillion**（朱红）- `#D94F30`（当前使用）
- **coral**（珊瑚）- `#E06B56`
- **teal**（青色）- `#2A7B9B`
- **amber**（琥珀）- `#D4A843`
- **forest**（森林绿）- `#2D8B55`

### 添加新模块

1. 在 `modules/` 目录中创建新的 HTML 文件
2. 在 `_base.html` 中添加对应的导航点
3. 重新运行 `bash build.sh`
