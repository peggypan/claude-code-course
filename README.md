# 🎓 Claude Code 交互式课程

这是 Claude Code 的交互式学习课程，包含两个版本，满足不同用户的需求。

---

## 📚 两个版本

### 🚀 入门指南（普通用户版）- 推荐新手

**文件**: [`course-beginner.html`](./course-beginner.html)

**适合人群**:
- 🌱 编程新手
- 💼 想快速上手的普通用户  
- 📚 需要向他人介绍 Claude Code 的人
- 🔍 不懂技术细节但想了解核心机制的用户

**课程特点**:
- **6个核心模块**，聚焦最重要概念
- **通俗易懂的语言**，避免技术术语
- **生活化类比**（餐厅点餐、智能管家等）
- **重点讲"是什么"和"为什么"**
- **包含实用技巧和避坑指南**

**课程大纲**:
1. Claude Code 是什么？
2. 如何与 Claude 对话
3. Claude 的工具箱
4. 为什么需要确认操作？
5. 个性化配置
6. 实用技巧和最佳实践

**学习时间**: 约 15-20 分钟

---

### 🎖️ 深入技术课程（工程师版）

**文件**: [`course-engineers.html`](./course-engineers.html)

**适合人群**:
- 👨‍💻 软件工程师和开发者
- 🔧 想深入了解 Claude Code 实现原理
- 📖 需要扩展 Claude Code 功能的高级用户
- 🔬 对 AI 应用架构感兴趣的技术人员

**课程特点**:
- **27个技术模块**，覆盖完整架构
- **198,000+ 行代码**的深度解析
- **详细的技术实现**和设计决策
- **架构图、流程图、状态机**可视化
- **代码示例和中文解释对照**

**课程大纲**（部分）:
1. CLI 入口与启动流程
2. REPL 核心循环
3. Query 执行引擎
4. 权限系统
5. Settings 配置
6. Agent 系统
7-14. ...更多模块
15-27. 高级特性（Bash 解析器、API 系统、认证等）

**学习时间**: 约 2-3 小时

## 📖 学习建议

### 对于完全新手

1. **先读入门指南** (`course-beginner.html`)
2. **动手实践** - 在实际项目中使用 Claude Code
3. **遇到问题时** - 再参考技术课程的对应章节

### 对于有经验的开发者

1. **快速浏览** 入门指南了解核心概念
2. **深入学习** 技术课程了解实现细节
3. **根据需要** 选择性阅读相关模块
