TypeScript 源码
async function* query(prompt) {
const messages = [];
let response = await api.call();
while (response.stop_reason) {
yield response;
response = await api.call();
}
}
中文逐行解释
定义一个可以生成多个结果的异步函数
创建一个空数组来存储对话历史
调用 Claude API 获取第一次响应
只要 AI 还需要使用工具，就继续循环
返回当前结果给用户（流式输出）
再次调用 API，传入工具执行结果
循环结束
函数定义结束

核心引擎：query 函数

这是 Claude Code 的心脏。 它不断地和 Claude API 对话，直到你的任务完成。

每次循环都像这样：AI 说"我需要用某个工具" → 系统执行工具 → 把结果告诉 AI → AI 继续思考下一步。

🎯 关键理解：

Claude Code 不是一次性给出答案，而是通过多次对话逐步完成任务。
