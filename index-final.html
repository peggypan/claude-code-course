Claude Code 是什么？

想象一下，你有一个超级聪明的程序员朋友，他能听懂你的话， 帮你写代码、查文件、修bug，甚至能解释整个项目是怎么运作的。

Claude Code 就是这样的朋友——一个运行在终端里的 AI 编程助手。

核心引擎：query 函数

这是 Claude Code 的心脏。

它不断地和 Claude API 对话，直到任务完成。

🎯 关键：

通过多次对话逐步完成任务，而不是一次性给出答案。
